Fhascards

Olá, hoje eu vou apresentar meu projeto de fhacards sobre filmes desenvolvido no Alura. O objetivo era criar uma ferramenta para ajudar na memorização de títulos, diretores e atores. Utilizei o Alura, GitHub e Vercel para desenvolver o projeto, que inclui criação de cartões, revisão aleatória e controle de progressos. Um dos desafios foi lidar com grande quantidade de dados, mas consegui superá-lo.Com esse projeto, aprendi mais sobre desenvolvimento web e consolidei minhas habilidades.O resultado foi uma ferramenta útil para mim e outros fãs de cinema.
